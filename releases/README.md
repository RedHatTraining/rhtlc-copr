# RHTLC Linux Release Assets

This directory contains Linux release binaries for RHTLC.

## Directory Structure

Each version has its own directory containing CLI and GUI binaries.

## Binary Information

- rhtlc-linux-x86_64: Command-line interface binary
- rhtlc-gui-linux-x86_64: Graphical user interface binary

## Platform Support

Built on AlmaLinux 8 (glibc 2.28), compatible with RHEL 8+, Fedora, AlmaLinux 8+, Rocky Linux 8+, CentOS Stream 8+
